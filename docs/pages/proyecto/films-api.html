Code

Acerca del proyecto

Actualmente en desarrollo

El proyecto es una API-REST que permite a través de un sistema de enrutamiento y peticiones HTTP administrar una colección de películas con toda la información asociada a ellas:

  • Título
  • Duración
  • Sinopsis
  • Trailers
  • Fotos
  • Directores
  • Actores/actrices
  • Guionistas

El objetivo final es crear una SPA con Angular 12 y comunicarla con esta API-REST para realizar una página similar a IMDb

Tecnologías empleadas

Films-API se ejecuta en un servidor creado con el framework Express, emplea MongoDB como base de datos no relacional y emplea el ODM Mongoose para la creación del CRUD.
