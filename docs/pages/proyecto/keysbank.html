Acerca del proyecto

KeysBank es una aplicación web de gestión de contraseñas diseñada tanto para uso privado cómo público.

Los usuarios con un estado 'activo' podrán registrar, editar y eliminar sus cuentas cuando lo requieran, beneficiándose de las herramientas que la app pone a su disposición cómo:

  • Un generador de contraseñas aleatorias que ofrece la contraseña más robusta posible para las características solicitadas.
  • Alertas de nick usados en otras cuentas registradas.
  • Alerta de contraseña demasiado antigua.
  • Capacidad de copiar al portapapeles la contraseña de una cuenta para un inicio de sesión rápido y cómodo.

En una aplicación de estas características es fundamental la seguridad. Por tanto, para garantizar un servicio robusto y evitar posibles filtraciones de información sensible la app se construye sobre 4 pilares:

  • Base de datos encriptada.
  • Protección ante posibles inyecciones SQL.
  • Consultas a la API para obtención dinámica de recursos o información (evitando el uso de información sensible en el árbol DOM).
  • Navegación y enrutamiento seguro según el perfil y los datos asociados.

Tecnologías empleadas

KeysBank se ejecuta en un servidor Apache, emplea PHP como lenguaje del backend y MySQL como base de datos. Del lado del frontend usa JavaScript Vanilla.

¿Qué puede hacer?

Gestiona cómodamente tus cuentas sin tener que depender de ningún servicio en la nube ☁️

  • Con una cuenta usuario cuyo estado sea "activo" podrás:
    • Agregar una cuenta propietaria y toda la información asociada a ella.
    • Editar una cuenta propietaria.
    • Eliminar una cuenta propietaria.
    • Beneficiarte de las herramientas que la app pone a tu disposición par ala gestión de tus cuentas propietarias.
  • Con una cuenta Administrador podrás:
    • Añadir, editar y eliminar plataformas y todas las cuentas asociadas a ellas asociadas a ellas.
      Nota: Una plataforma es todo aquella web, servicio y/o software donde se puedan registra cuentas. Ejemplo: Netflix, Amazon, PayPal, Ubuntu, etc
    • Activar, bloquear y eliminar usuarios.
