Acerca del proyecto

El juego del Buscaminas, juego clásico incorporado en Windows.

Configura la difultad del juego y ¡¡trata de batir tu proio record!!

Este proyecto fue entregado como un ejercicio de clase y se trata de una mejora de este otro proyecto personal: Antiguo buscaminas

Tecnologías empleadas

Para este proyecto únicamente he necesitado emplear HTML5 como lenguaje de marcas, CSS para definir las hojas de estilos y JavaScript Vanilla para la lógica del juego.
