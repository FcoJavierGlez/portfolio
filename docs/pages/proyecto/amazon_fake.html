Acerca del proyecto

Interfaz gráfica de una tienda diseñada como Amazon con un carrito de la compra funcional. Actualmente no posee un backend, todos los elementos del proyecto son cargados desde arrays.

Este proyecto fue creado para aprender acerca de los componentes de React su ciclos de vida asociados a sus estados y comportamientos.

Tecnologías empleadas

Para este proyecto úncamente he necesitado emplear HTML5 como lenguaje de marcas, CSS para definir las hojas de estilos y JavaScript Vanilla para la lógica del juego.

¿Qué puede hacer?

Simular un carrito de la compra. Recuerda comprar con moderación. 😎
