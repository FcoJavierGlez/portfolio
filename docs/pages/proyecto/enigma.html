Acerca del proyecto

Programa creado en Java-8 capaz de cifrar y descifrar tanto strings como ficheros empleando una llave que puede ser generada aleatoriamente o importada de una generación anterior.

Cada llave consta de 7 capas de generación aleatoria. Lo que otorga de una gran robustez al proceso de cifrado.

Este proyecto fue creado en mi primer año de programación.

Tecnologías empleadas

Este fue mi primer proyecto. Para su realización utilicé Java-8 que es uno de los dos lenguajes de programación con los que me inicié en el mundo del desarrollo.

¿Qué puede hacer?

Cifrar / descifrar información usando una llave generada con 7 capas de aleatoriedad.

Una vez generada una llave puedes importar un archivo de texto y cifrarlo con esa llave. A continuación el fichero cifrado podrá ser descifrado usando la misma llave. A éste proceso de cifrado / descifrado usando una misma clave (llave) se le denomina sistema de cifrado simétrico.
