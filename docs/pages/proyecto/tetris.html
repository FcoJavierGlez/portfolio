Acerca del proyecto

El videojuego de Tetris clásico creado con fines autodidactas. Lo que pretendía cuando desarrollé este juego era experimentar con características actualmente en desarrollo de JavaScript, particularmente con la privatización de métodos y atributos usando el signo '#'. Ejemplo: this.#name.

Y de paso, ¿porqué no decirlo?, poder disfrutar y compartir mi propio Tetris que siempre está bien. 😉

Importante: estas características (a fecha de redactar este artículo) sólo permiten jugar en navegadores Chromium: Chrome, Ópera, Edge, etc.

Tecnologías empleadas

Para este proyecto úncamente he necesitado emplear HTML5 como lenguaje de marcas, CSS para definir las hojas de estilos y JavaScript Vanilla para la lógica del juego.
